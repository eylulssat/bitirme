🚀 KURULUM ADIMLARI:

1️⃣ VERİTABANI (pgAdmin4):
   - bebook veritabanını aç
   - Query Tool'u aç
   - backend/create_favorites_table.sql dosyasını çalıştır

2️⃣ BACKEND:
   cd bebook/backend
   uvicorn main:app --reload --host 0.0.0.0 --port 8000

3️⃣ FLUTTER:
   cd bebook
   flutter pub get
   flutter run

🔗 API ENDPOINT'LERİ:

POST   /favorites/toggle              → Favorilere ekle/çıkar
GET    /favorites/{user_id}           → Favorileri listele
GET    /favorites/check/{user_id}/{book_id} → Favori kontrolü
